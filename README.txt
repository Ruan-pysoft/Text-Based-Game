How to play:

1:
Install python 3.6 or higher (I used 3.7 to write the game)
Windows:
    Go to https://www.python.org/downloads/
    Select a Python version
    Scroll down to "Files"
    Download and run one of the windows installers
Linux:
    This guide worked for me: https://linuxize.com/post/how-to-install-python-3-7-on-ubuntu-18-04/

2:
Windows:
    Run "game.bat"
Linux:
    Run "game.sh"
